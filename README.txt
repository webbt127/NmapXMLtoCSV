Prerequisites:
    Python 3.11 installed

Instructions:
    Run setup.bat to install required packages
    On Windows, run "script.bat" to start the converter GUI
    Choose a file to convert and an output destination
    Optionally, instead of the GUI you can pass the paths as arguments though Windows CMD using optional flags --file_path & --folder_path
        Example: /path/to/project/python main.py --file_path C:/Users/POWER Engineers/Documents/MyXML.xml --folder_path C:/Users/POWER Engineers/Documents/
